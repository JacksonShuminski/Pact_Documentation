Heavy Projectile

The light projectile is a Projectile that inherits behaviors from Base Projectile. It uses an Abilities Struct to determine if it has upgrades relevant to heavy attacks before acting out any of the upgrade behaviors. The OnCreation event has logic in place to damage the player if the shot was fired by the player.

Vulnerable Debuff Upgrade

The vulnerable upgrade increases damage to the player and enemy as stacks build up. The implementation of the Vulnerable effect is done in Base Character, specifically in the DamageCharacter event. The projectile uses the AddVulnerable method on the Base Character in the OnHit method by the amount defined in the Ability Struct.
